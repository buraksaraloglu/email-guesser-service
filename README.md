## How to start?

```zsh
# Install dependencies
pnpm i

# Activate git hooks
pnpm prepare

# Start development
pnpm dev

# Build production code
pnpm build

# Run production code
node build

# New commit with interactive CLI
pnpm cz

# Auto generate changelogs and versioning
pnpm release
```
